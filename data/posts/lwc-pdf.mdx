## Steps

- Add static resource of latest [jsPDF library](https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js) 
library to enable for its usage. Save the page as js extension and add it to the static resource of your org.
You can do it by adding it through the the org interface or drag the file to the static resource of your org in VS Code.
- Create a Lightning Web Component in your project. It is up to you to make it as a quick action, normal button or etc.
- Refer to the snippets of [jsPDF: Generate or save PDF in LWC](/snippets/jspdf-generate-or-save-pdf-in-lwc) for example

Feel free to add or edit the code. Eg: Preview generated PDF in new window
